# Инструкция по запуску

# Переходим в директорию server
* cd server
  
# Устанавливаем зависимости
* npm install
  
# Запускаем сервер
* npm run start

# Переходим в директорию client в новой вкладке терминала
* cd client

* # Устанавливаем зависимости
* npm install

# Запускаем клиента
* npm run start
